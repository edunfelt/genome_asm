DataTrim class

What it does:

  • Input: text file in which each row contains 12 values, according to the specifiations in the project instructions
  • Output: text file in which value 3, 4, 5 and 9 on each row has been removed, the values at place 1 and 2 has been changed to integers, and rows that represent complete overlaps are removed

Private methods:

  • Main: reads file line by line and removes unneccessary columns, and changes the id's to an integer value starting at 0, utilizes the private method Unique to identify and ignore rows that describe complete overlaps. Writes to new file.
  • Unique(String s): takes a string array and returs True if it describes a complete overlap, False otherwise
